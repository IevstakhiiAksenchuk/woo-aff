== Description ==

Simple Affiliate plugin for WooCommerce by eMagicOne provides affiliates with the opportunity to earn income by sharing affiliate links or/and discounts. One-time payment, no recurring fees. WPML and Multi-store support available. The plugin is offered as is.

Key features include:
– Affiliate Dashboard: Enables affiliates to track their link clicks, performance and earnings in real time.
– Admin Management: Allows administrators to manage affiliate accounts, set commission rates, and view reports.
– Manual Payouts: Facilitates easy handling of affiliate payouts.
– Real-time Tracking: Tracks affiliate referrals, sales, and commissions in real-time.

== Third-party services ==

== Installation ==

1. Upload the `emagicone-affiliate-for-woocommerce` folder to your WordPress plugin directory (`/wp-content/plugins/`).
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the plugin settings to configure your affiliate program parameters.
Installation and configuration instructions can be found at: woocommerce-affiliate.emagicone.com

== Frequently Asked Questions ==

= How do I set up the affiliate program? =

After activating the plugin, go to the plugin settings where you can configure the commission rates and other settings related to your affiliate program.

= Can affiliates track their performance? =

Yes, the plugin provides a dashboard for affiliates where affiliates can track their performance, and view earnings.

= Installed plugin and cannot see affiliate page? =

Go to Settings -> Permalinks and hit "Save Changes"
